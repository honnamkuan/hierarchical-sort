# HierarchicalSort

Example to sort an hierarchical list of object with weightage to achieve the desired sorted outputs:

```
root [weight(w)=350]  
│  
├ b [w=250]  
│ ├ b.2 [w=150]  
│ │  
│ ├ b.3 [w=99]  
│ │  ├ b.32 [w=50]  
│ │  └ b.31 [w=49]  
│ │  
│ └ b.1 [w=1]  
│  
└ a [w=100]  
  ├ a.1 [w=80]  
  └ a.2 [w=20]
```
